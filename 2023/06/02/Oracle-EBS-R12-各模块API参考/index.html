Oracle iby_fndcpt_common_pub

Oracle IBY_FNDCPT_COMMON_PUB是一个Oracle E-Business Suite的应用程序编程接口（API），它是Oracle Payments模块中的一部分。该API包含一些公共功能，可用于与IBY_FM_GLOBAL_PKG一起完成诸如支付令牌处理、验证银行账户、验证信用卡信息等支付相关的任务。由于它是公共API，因此可以在任何Oracle E-Business Suite应用程序模块中使用它。

Oracle iby_ext_bankacct_pub

Oracle iby_ext_bankacct_pub是Oracle E-Business Suite中的一个API类，用于管理付款凭证的银行账户信息。它提供了几个方法，包括创建、更新和删除银行账户信息以及验证银行账户信息等操作。

使用这个API类，可以节省在其他系统中手动输入和维护银行账户信息的时间和精力，它可以与其他模块集成，比如付款，方便用户轻松管理付款凭证的银行账户信息。

需要注意的是，使用这个API类需要有相关权限和正确的配置，建议在使用API之前仔细阅读相关的文档和指南。
